# Berber Cement KPI Dashboard

An interactive Streamlit dashboard for Berber Cement plant KPIs, maintenance tracking, and ML-based predictive analytics for VRM and Kiln operations.

## Features

- **KPI Dashboard**: View and edit KPIs across departments (Production, Maintenance, Utility, etc.)
- **Real-time Editing**: Edit actual values directly in tables and save to Excel
- **Status Tracking**: Automatic status calculation (On Track, Attention, Critical)
- **ML Models**:
  - VRM Advanced ML: Predict residue, trip risk, and stability score
  - Kiln ML: Predict free lime, ring risk, and stability score
- **Data Visualization**: Interactive charts for target vs actual comparison
